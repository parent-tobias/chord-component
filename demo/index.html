API Documentation

chord-diagram

Displays a single chord diagram.

  • chord (string): The chord name (e.g., "C", "Am7", "F#dim")
  • instrument (string): The instrument type (default: "Standard Ukulele")

chord-list

Displays multiple chord diagrams in a responsive grid.

  • instrument (string): The instrument type (default: "Standard Ukulele")
  • chords (string|array): JSON string or array of chord names

Supported Instruments

  • Standard Ukulele
  • Baritone Ukulele
  • 5ths tuned Ukulele
  • Standard Guitar
  • Drop-D Guitar
  • Standard Mandolin
